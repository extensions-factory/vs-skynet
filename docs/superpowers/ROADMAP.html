Skynet — Product Roadmap

Skynet turns CLI coding agents (Codex CLI, Claude Code, Antigravity) into a real Scrum team — the developer as Scrum Master. Every agent is a worker = model + role + soul, driven interactively through a visible terminal, never a headless flag or SDK.

Three levels: Epics (cards below), the Features in each, and the User Stories under each feature. Filter by status with the legend. Narrative vision, release plan, and risk gates live in ROADMAP.md; architecture in ARCHITECTURE.md.
